# SkillTracking---Management-System-for-tutor-platform
web application module that enables tutor discovery, skill tracking, tutor profile management, and advanced filtering by subject, level, availability, and price range.
 1️⃣ Skill Tracking
- Monitor student learning progress
- Track completed lessons
- Calculate skill improvement
- Store progress history
 
2️⃣ Tutor Discovery
Students can:
- Browse tutors by STEM / ICT subjects
- Search tutors by subject name
- Filter tutors by:
  - Teaching Level (School / University)
  - Availability
  - Price Range
- View detailed tutor profiles including:
  - Qualifications
  - Subject expertise
  - Experience
    
 3️⃣ Tutor Management
Tutors can:
- Add their profile
- Update profile information
- Manage subject listings
- Remove their profile if needed
